# AI Fitness & Diet Planner

A premium, AI-powered progressive web app for tracking fitness, diet, and social streaks. Built with React, Vite, and a custom Glassmorphism UI.

## 🚀 Features

- **AI Calorie Tracking**: Snap a photo of your food to estimate calories (simulated AI).
- **Smart Workouts**: Personalized workout plans based on your goals (Muscle Gain, Weight Loss, Maintenance).
- **Social Implementation**: Connect with friends via unique codes and track streaks.
- **Glassmorphism UI**: modern, dark-mode first design with smooth animations.
- **Local First**: Data persists in your browser's LocalStorage.

## 🛠️ Setup & Installation

> **Note**: This project requires [Node.js](https://nodejs.org/) to be installed on your machine.

1.  **Install Node.js**
    - Download and install from [nodejs.org](https://nodejs.org/).
    - Verify installation by running `node -v` and `npm -v` in your terminal.

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```

4.  **Build provided for Production**
    ```bash
    npm run build
    ```

## 📱 Usage

1.  **Onboarding**: Complete the initial profile setup.
2.  **Dashboard**: proper overview of daily stats.
3.  **Diet**: Use the camera icon to log meals.
4.  **Fitness**: Check off daily exercises.
5.  **Social**: Share your unique code with friends to compete.
